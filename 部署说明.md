# ETLink 官网部署说明

## 文件结构
```
etlink/
├── index.html          # 完整单页官网（含微信二维码base64内嵌）
├── api/
│   └── leads.js        # 表单API（Google Sheets + SendGrid）
├── vercel.json         # Vercel部署配置
├── package.json        # 项目依赖
└── 部署说明.md
```

## 快速部署步骤

### 1. 安装并登录 Vercel
```bash
npm install -g vercel
vercel login
```

### 2. 进入项目目录并部署
```bash
cd etlink
vercel --prod
```

### 3. 配置环境变量（Vercel Dashboard → Settings → Environment Variables）

| 变量名 | 值 | 说明 |
|---|---|---|
| `SENDGRID_API_KEY` | `SG.xxxxx` | SendGrid API Key |
| `NOTIFY_EMAIL` | `hello@etlink.com.au` | 通知收件邮箱 |
| `FROM_EMAIL` | `noreply@etlink.com.au` | 发件邮箱（需SendGrid认证） |
| `GOOGLE_SHEETS_WEBHOOK_URL` | `https://script.google.com/...` | Google Apps Script URL |

## Google Sheets 配置

1. 新建 Google Sheets，第一行设置列标题：`时间|姓名|公司|邮箱|岗位|需求描述|来源`
2. 打开「扩展程序 → Apps Script」，粘贴以下代码：

```javascript
function doPost(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  const data = JSON.parse(e.postData.contents);
  sheet.appendRow([data.timestamp, data.name, data.company,
    data.email, data.position, data.message, data.source]);
  return ContentService.createTextOutput(JSON.stringify({success:true}))
    .setMimeType(ContentService.MimeType.JSON);
}
```

3. 点击「部署 → 新建部署」，类型选「网页应用」，权限选「任何人」
4. 复制生成的 URL 填入环境变量 `GOOGLE_SHEETS_WEBHOOK_URL`

## SendGrid 配置

1. 注册 [sendgrid.com](https://sendgrid.com)
2. Settings → API Keys → Create API Key（开启 Mail Send 权限）
3. 完成发件域名认证（Sender Authentication）

## 绑定自定义域名

Vercel Dashboard → 项目 → Settings → Domains → 输入域名按提示添加DNS记录
